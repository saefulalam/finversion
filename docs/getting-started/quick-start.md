# Quick Start

## 1. Add to HTML

```html
<link rel="stylesheet" href="dist/styles/fv-ui.css">
```

## 2. Use components

```html
<button class="fv-btn fv-btn--primary">Click me</button>

<div class="fv-card">
  <h3>Card Title</h3>
  <p>Card content</p>
</div>
```

## 3. Toggle Dark Mode

```html
<html data-fv-theme="dark">
```

## 4. JavaScript API

```javascript
FV.toast('Hello!', 'success')
FV.modal('my-modal').open()
```
